# Q and A

### How to do things

1. clone repo
2. cd into `qanda`
3. `npm install`
4. `npm run up`
5. _optional:_ `npm run seed`
6. `npm start`
7. [add questions](http://localhost:3000/admin)
8. [answer questions](http://localhost:3000)
